# Métodos de Cadenas en JavaScript


### Obtener la Longitud
El método `length` devuelve el número total de caracteres en una cadena.
```javascript
var texto = "Esto es una oracion";
var longitud = texto.length;
console.log("Longitud:", longitud);
```

### Cortar Cadenas

#### substring()
Corta una cadena desde una posición específica hasta el final.
```javascript
var texto = "Esto es una oracion";
var textosubstring = texto.substring(5); // Corta desde la posición 5
console.log("Texto cortado:", textosubstring);
```

#### substr()
Corta una cadena especificando inicio y longitud.
```javascript
var textosubstr = texto.substr(0, 8); // Desde posición 0, 8 caracteres
console.log("Texto cortado:", textosubstr);
```

### Búsqueda de Caracteres

#### indexOf()
Encuentra la primera ocurrencia de un carácter o subcadena (busca de inicio a fin).
```javascript
var posicion = texto.indexOf("o", 0); // Busca desde la posición 0
console.log("Primera posición de 'o':", posicion);
```

#### lastIndexOf()
Encuentra la última ocurrencia de un carácter o subcadena (busca de fin a inicio).
```javascript
var posicionlast = texto.lastIndexOf("o");
console.log("Última posición de 'o':", posicionlast);
```

### Modificación de Texto

#### replace()
Reemplaza una parte del texto por otra.
```javascript
var nuevotexto = texto.replace("una oracion", "un texto");
console.log("Texto reemplazado:", nuevotexto);
```

### Cambio de Mayúsculas y Minúsculas

#### toUpperCase()
Convierte todo el texto a mayúsculas.
```javascript
var textoupper = texto.toUpperCase();
console.log("Texto en mayúsculas:", textoupper);
```

#### toLowerCase()
Convierte todo el texto a minúsculas.
```javascript
var textolower = texto.toLowerCase();
console.log("Texto en minúsculas:", textolower);
```

## Consejos Prácticos
- Los métodos de cadenas no modifican la cadena original
- La posición de los caracteres comienza desde 0
- Los métodos son sensibles a mayúsculas y minúsculas
