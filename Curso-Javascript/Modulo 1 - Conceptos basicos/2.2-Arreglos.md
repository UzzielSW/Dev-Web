# Arreglos en JavaScript

## Declaración de Arreglos

```javascript
var cursos1 = ["HTML", "CSS", "JS"];
```

> **Nota**: Si intentas acceder a un índice fuera del rango del arreglo, obtendrás `undefined`.

## Métodos Básicos de Arreglos

### Acceso a Elementos

```javascript
var family = ["Family"];
console.log("Primer valor:", family[0]);
```

### Longitud del Arreglo

```javascript
console.log("Cantidad de elementos:", family.length);
```

### Agregar Elementos

```javascript
family.push("Wester", "Fermin", "Moises");
```

### Eliminar Elementos

```javascript
family.pop(); // Elimina el último elemento
```

### Unir Arreglos

```javascript
var Others = ["Cristina", "Ashly", "Leonardo"];
family.concat(Others);
```

### Convertir a Cadena

```javascript
family.join(" <=> "); // Une elementos con un separador
```

### Ordenamiento

```javascript
family.sort(); // Ordena ascendentemente
family.reverse(); // Ordena descendentemente
```

## Métodos Avanzados de Arreglos

### map()

Transforma cada elemento del arreglo según una función callback.

```javascript
let estudiantes = [
  { nombre: "Carlos", calificacion: 20 },
  { nombre: "Juan", calificacion: 12 },
  { nombre: "Maria", calificacion: 9 },
  { nombre: "Pepe", calificacion: 7 },
  { nombre: "Arturo", calificacion: 17 },
];

// Obtener solo los nombres
let estudiantesNombres = estudiantes.map((estudiante) => estudiante.nombre);

// Ejemplo con números
let numbers = [1, 2, 3, 4, 5];
let doubles = numbers.map((number) => number * 2); // [2, 4, 6, 8, 10]
```

### filter()

Filtra elementos según una condición.

```javascript
let estudiantesAprobados = estudiantes.filter(
  (estudiante) => estudiante.calificacion > 10
);
```

### reduce()

Reduce el arreglo a un solo valor.

```javascript
let numeros = [2, 4, 5, 60, 8, 10];

// Suma de elementos
let suma = numeros.reduce((a, b) => a + b);

// Encontrar máximo
let max = numeros.reduce((a, b) => (a > b ? a : b));

// Calcular promedio
let promedio = numeros.reduce((a, b, i, arr) => {
  b += a;
  return i == arr.length - 1 ? b / arr.length : b;
});
```

## Matrices

```javascript
let nEcu = 3; // número de filas
let nVariable = 3; // número de columnas

// Crear matriz
let sistema = new Array(nEcu);
for (var i = 0; i < sistema.length; i++) {
  sistema[i] = new Array(nVariable);
}

// Inicializar valores
for (var i = 0; i < sistema.length; i++) {
  for (var k = 0; k < sistema[i].length; k++) {
    sistema[i][k] = 0.0;
  }
}
```

## Consejos Prácticos

- Los métodos de arreglos pueden encadenarse
- Algunos métodos modifican el arreglo original, otros crean uno nuevo
- Usa `map()`, `filter()` y `reduce()` para operaciones funcionales

> **Nota**: Los métodos modernos como `map()`, `filter()` y `reduce()` son preferibles a los bucles tradicionales para operaciones de transformación de datos.
