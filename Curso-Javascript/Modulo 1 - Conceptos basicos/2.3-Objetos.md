# Objetos en JavaScript

## Declaración de Objetos

### Objeto Literal

```javascript
var person = {
  name: "julio",
  age: 31,
  height: 182,
};
```

## Acceso a Propiedades

```javascript
// Notación de punto
console.log(person.age); // 31

// Notación de corchetes
console.log(person["name"]); // julio
```

## Métodos en Objetos

### Método Simple

```javascript
var robot = {
  name: "android",
  saludar() {
    console.log(`Hello, I am ${this.name}`);
  },
};

// Llamar al método
robot.saludar();
```

## Constructores de Objetos

### Constructor Básico

```javascript
function objPerson(name, age) {
  this.name = name;
  this.age = age;

  this.changeName = (name) => {
    this.name = name;
  };
}

// Crear una nueva instancia
var persona1 = new objPerson("David", 21);
console.log(`Name persona1: ${persona1.name}`);

// Cambiar nombre usando el método
persona1.changeName("Jonh");
console.log(`New name persona1: ${persona1.name}`);
```

### Constructor con Métodos Externos

```javascript
function objPersonPlus(name, age) {
  this.name = name;
  this.age = age;
  this.yearOfBirth = bornYear;
}

function bornYear() {
  return 2025 - this.age;
}

var persona2 = new objPersonPlus("User", 19);
console.log(`${persona2.name} nacio en el year ${persona2.yearOfBirth()}`);
```

## Combinación de Objetos

### Spread Operator

```javascript
const user = {
  name: "KD",
  last: "Root",
};

const userInfo = {
  age: 18,
};

// Combinar objetos
const result = {
  ...user,
  ...userInfo,
};
```

## Conceptos Importantes

### Palabra Clave `this`

- `this` se refiere al objeto actual dentro de un método
- En funciones flecha, `this` mantiene el contexto del objeto que lo contiene
- En métodos regulares, `this` puede cambiar según cómo se llame la función

### Buenas Prácticas

- Usa nombres descriptivos para las propiedades y métodos
- Mantén la consistencia en el estilo de nombrado
- Considera usar clases para objetos más complejos
- Utiliza el spread operator para crear copias de objetos
